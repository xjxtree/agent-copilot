# V2.69 Prototype Notes

V2.69 is a production polish slice rather than a new surface. The design uses existing Settings and detail panels:

- Settings gains a screenshot privacy mode toggle in the Language tab.
- High-risk path rows render with screenshot-safe placeholders and middle collapse by default.
- Full path reveal is explicit, local to the current view, and not persisted.
- Task Cockpit remains the first detail surface so privacy changes can be inspected on the default screen.
- Capture validation rejects invalid images before writing screenshot evidence.
