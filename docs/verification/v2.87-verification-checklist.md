## Scope

- Product display name is Agent Copilot while internal `SkillsCopilot` /
  `skills-copilot` identifiers remain stable.
- Lineup default surface summarizes agent readiness, risk, cleanup pressure,
  task/provider context, and evidence-backed navigation.
- Agent Profile shows per-agent health, capability, scan status, MCP preview,
  and nearby evidence surfaces.
- `session.previewLocalSessions` reads only explicitly authorized local session
  directories and returns redacted excerpts/evidence refs.

  drift verification.
