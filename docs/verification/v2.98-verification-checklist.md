# V2.98 Verification Checklist

Status: completed

## Scope

V2.98 closes Automatic local session discovery and session-derived skill usage
summaries for the current Agent Copilot Sessions/Skills workflow.

- [x] `session.previewLocalSessions` auto-discovers supported Claude Code, Codex, opencode, and Pi local session stores when no explicit authorized roots are supplied.
- [x] The response returns redacted `session_rows`, bounded filterable
  `content_items`, and session-derived `skill_usage_rows`.
- [x] Skill usage counts are derived from explicit local invocation markers such
  as `/skill`, `/skill:<name>`, and `skill:<name>`, not ordinary skill-name
  mentions.
- [x] Tool calls and skill calls remain separate counters from user/agent/thinking
  message totals.
- [x] Hermes/OpenClaw session parsing remains deferred until their session-store
  formats have confirmed evidence.
- [x] No credential read, credential write, credential export, or credential
  handling expansion was added.
- [x] No provider default call, write path, script execution, credential access,
  raw transcript persistence, cloud sync, telemetry, signing, packaging, DMG,
  ZIP, notarization, or release automation was added.

## Safety Decision

Final status decision: completed.

V2.98 is read-only local evidence discovery. It does not persist raw transcript
content, raw prompts, raw responses, secrets, or unredacted paths, and it does
